## 🛠️ The Build (Touch-by-Touch)

This project was architected for maximum visual impact and technical SEO performance. Here is a breakdown of the core systems:

### 1. The Visual Identity (Comic-Tech Aesthetic)
*   **Design Language:** Uses a high-contrast, "editorial" layout inspired by modern tech magazines and comic books.
*   **Typography:** Leverages the `Bangers` font for high-impact headings and `Inter` for technical readability.
*   **Color Palette:** A curated slate-gray background contrasted with vibrant "Cyber Lime" (`#b8ff00`) and "Comic Pink" (`#ff0066`).

### 2. The 3D Engine (React Three Fiber)
*   **Interactive Avatar:** A high-fidelity 3D model managed via `Three.js` and `@react-three/fiber`.
*   **Cinematic Lighting:** Custom studio-grade lighting setup including a warm Key light, cool Fill light, and a dedicated Rim light for depth.
*   **Subsurface Scattering:** Simulated via subtle emissive properties on the skin materials for a lifelike, non-plastic appearance.
*   **Universal Framing:** Dynamic Field of View (FOV) and camera positioning that automatically adjusts between mobile (circular portrait) and desktop (cinematic wide-shot).

### 3. Technical Foundation
*   **React + Vite:** For ultra-fast development and optimized production bundles.
*   **TypeScript:** Ensuring type-safety across the 3D scene and data structures.
*   **Tailwind CSS:** Custom design system using utility-first classes for rapid, responsive UI development.
*   **Formspree Integration:** A fully functional, serverless contact form backend.

### 4. SEO & Visibility
*   **JSON-LD Person Schema:** Integrated structured data to help search engines establish a Google Knowledge Panel.
*   **Entity Resolution:** Connects the portfolio to official profiles on Wikidata, Crunchbase, Google Scholar, and ResearchGate.
